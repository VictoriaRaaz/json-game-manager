# JSON Game Manager
Aplicación desarrollada en Java que permite gestionar un conjunto de videojuegos utilizando un archivo JSON como base de datos documental.

## Descripción
Este proyecto consiste en un gestor de videojuegos con interfaz gráfica desarrollada con Java Swing, que permite realizar operaciones (como Añadir, editar, buscar y eliminar) sobre los datos almacenados en un único archivo JSON.

## Tecnologías utilizadas
- Java
- Java Swing (interfaz gráfica)
- Gson (gestión del JSON)
- Maven (gestión de dependencias)
